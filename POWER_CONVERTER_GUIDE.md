## 一、电源类题型概述

### 1.1 四大变换类型

| 类型 | 定义 | 典型应用 | 电赛高频题 |
|------|------|----------|-----------|
| **DC-DC** | 直流转直流 | Buck降压、Boost升压、Buck-Boost | ⭐⭐⭐⭐⭐ |
| **DC-AC** | 直流转交流 | 逆变器、UPS、SPWM | ⭐⭐⭐⭐ |
| **AC-DC** | 交流转直流 | 整流器、PFC、充电器 | ⭐⭐⭐ |
| **AC-AC** | 交流转交流 | 变压器、调压器、矩阵变换器 | ⭐⭐ |

### 1.2 电赛电源题核心能力要求

1. **效率** — 通常要求 ≥ 85%~95%
2. **纹波** — 输出电压纹波 ≤ 50mVpp
3. **负载调整率** — 空载→满载电压变化 ≤ 1%
4. **电压调整率** — 输入电压波动时的输出稳定性
5. **动态响应** — 突加/突卸负载的恢复时间
6. **保护功能** — 过压、过流、过热、短路保护

## 二、DC-DC 变换器

### 2.1 Buck 降压变换器

**原理：** 输入高压→开关管通断→LC滤波→输出低压。

$$V_{out} = D \times V_{in} \quad (D = 占空比)$$

**关键参数：**
- 开关频率：通常 50kHz~200kHz
- 电感取值：$$L = \frac{V_{out} \times (1-D)}{f_s \times \Delta I_L}$$
- 输出电容：$$C_{out} = \frac{\Delta I_L}{8 \times f_s \times \Delta V_{out}}$$

**电赛考点：**
- 高效率同步Buck（用MOSFET代替二极管）
- 多相Buck（大电流场合）
- 宽输入范围Buck（如 7V~30V 输入）

### 2.2 Boost 升压变换器

**原理：** 输入低压→电感储能→释放到输出端→升压。

$$V_{out} = \frac{V_{in}}{1-D}$$

**关键参数：**
- 电感取值：$$L = \frac{V_{in} \times D}{f_s \times \Delta I_L}$$
- 输出电容主要取决于输出纹波要求
- **注意：** Boost不能空载运行（输出会失控升高）

**电赛考点：**
- PFC（功率因数校正）前置Boost
- 光伏升压MPPT

### 2.3 Buck-Boost / SEPIC / Cuk

- **Buck-Boost：** 输出可升可降，但极性反转
- **SEPIC：** 输出可升可降，极性不变，需要两个电感
- **Cuk：** 输出可升可降，纹波小，但电路复杂

**电赛策略：** 大部分题目用 Buck 或 Boost 即可，SEPIC 很少考。

### 2.4 DC-DC 控制方式

| 控制方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| **电压控制** | 简单、单环 | 响应慢 | 负载变化小的场合 |
| **电流控制** | 响应快、限流自然 | 需要电流采样 | 动态响应要求高 |
| **平均电流控制** | 精度高 | 复杂 | PFC、高精度电源 |
| **V² 控制** | 超快瞬态响应 | 噪声敏感 | 处理器供电 |

### 2.5 常用芯片

- **集成MOSFET：** TPS5430, LM2596, MP2307
- **外置MOSFET控制器：** TL494, SG3525, UC3842/3/4/5
- **同步整流控制器：** ISL8120, TPS40057

## 三、DC-AC 逆变器

### 3.1 基本原理

DC-AC 逆变器将直流电转换为交流电。核心思路是用 PWM 调制出一个正弦波。

$$v_{out}(t) = V_m \times \sin(\omega t)$$

### 3.2 SPWM（正弦脉宽调制）

**原理：** 用正弦波与三角载波比较，产生宽度按正弦规律变化的脉冲。

**关键参数：**
- 载波频率：通常 10kHz~50kHz
- 调制比：$$m = \frac{V_{正弦波峰值}}{V_{三角波峰值}} \leq 1$$
- 输出基波幅值：$$V_{o1} = m \times V_{dc}$$

**单极性SPWM vs 双极性SPWM：**

| 类型 | 谐波含量 | 开关损耗 | EMI |
|------|----------|----------|-----|
| 单极性 | 低 | 中 | 低 |
| 双极性 | 高 | 高 | 高 |

### 3.3 全桥逆变器

**经典拓扑：** 4个MOSFET构成H桥，对角线交替导通。

**驱动时序：**
```
Q1/Q4导通 → 正向输出
死区时间
Q2/Q3导通 → 反向输出
```

**电赛考点：**
- 死区设置（防止直通炸管）
- LC滤波设计（滤除开关频率分量）
- 输出调压（调制度控制）

### 3.4 常见题型

- **单相逆变器：** 输出 220V/50Hz 正弦波
- **三相逆变器：** 三路互差120°的SPWM
- **在线式UPS：** AC→DC→AC 两级变换
- **变频器：** DC→AC 频率可调

### 3.5 控制策略

- **PID 控制输出电压有效值**
- **PR 控制（比例谐振）**：对正弦波无静差跟踪
- **重复控制**：周期性的负载扰动补偿

## 四、AC-DC 整流器

### 4.1 不可控整流

**二极管整流桥 + 滤波电容：**

最简单的方式，但：
- 电流畸变严重（THD > 100%）
- 功率因数低（约 0.6）
- 只能调压不能稳压

### 4.2 相控整流（可控硅）

**晶闸管移相触发控制：**

$$V_{out} = V_m \times \frac{1+\cos\alpha}{2}$$

- $$\alpha$$ 为触发角
- 0°<α<180°
- 调压范围大，但谐波含量高

### 4.3 PWM 整流器（PFC）

**有源功率因数校正（APFC）：**

| 类型 | 工作模式 | 适用功率 | 复杂度 |
|------|----------|----------|--------|
| 临界模式 (CrM) | 变频 | < 300W | 低 |
| 连续模式 (CCM) | 定频 | > 300W | 中 |
| 交错式 | 两路交错 | > 500W | 高 |

**PFC 控制目标：**
1. 输入电流跟随输入电压（单位功率因数）
2. 输出稳定的直流电压
3. 低谐波（THD < 5%）

### 4.4 常用PFC控制芯片

- UCC28019, UCC28180
- L6562, L6563
- NCP1654

### 4.5 电赛考点

- **AC-DC 充电器：** AC 220V→DC 可调输出
- **PFC 前置级：** 提高功率因数至 0.99
- **隔离型 AC-DC：** 反激变换器（Flyback）

## 五、AC-AC 变换器

### 5.1 传统方案：变压器 + 调压

- 自耦调压器（手动旋钮）
- 晶闸管调压（斩波调压）

**优缺点：** 简单、可靠，但体积大、效率低、谐波高。

### 5.2 矩阵变换器

**直接 AC-AC 变换，无中间直流环节：**

- 双向开关阵列
- 输出电压和频率均可调
- **优点：** 能量双向流动、功率密度高
- **缺点：** 控制复杂、器件多

### 5.3 交流稳压电源

**电赛常见题型：**
- 输入电压波动时维持输出电压稳定
- 通常采用 AC→DC→AC 间接变换
- 实际上是用逆变器（DC-AC）实现的

### 5.4 电赛中的 AC-AC

**策略：** 电赛中纯 AC-AC 题目极少。遇到"交流稳压"类题目，建议：
1. 先用变压器降压
2. 然后 AC→DC→AC 两级处理
3. 用 Buck/Boost 稳定中间直流母线

## 六、电赛电源题通用设计方法

### 6.1 设计流程

```
需求分析 → 拓扑选择 → 参数计算 → 仿真验证
    → 硬件设计 → 开环调试 → 闭环调试 → 极限测试
```

### 6.2 关键模块通用设计

#### 功率器件选择

| 器件 | 耐压余量 | 电流余量 | 散热考虑 |
|------|----------|----------|----------|
| MOSFET | 2×V_max | 3×I_rms | 加散热片 |
| 二极管 | 2×V_RRM | 2×I_avg | 选肖特基 |
| 电感 | — | 1.2×I_peak | 磁芯不饱和 |
| 电容 | 1.5×V_rated | — | 低ESR |

#### 驱动电路

- **高端驱动：** 自举电路或隔离驱动
- **低端驱动：** 直接驱动或图腾柱
- **死区时间：** 50ns~200ns

#### 采样电路

| 采样对象 | 方法 | 注意事项 |
|----------|------|----------|
| 输出电压 | 电阻分压 + 运放 | 隔离？共模？ |
| 输出电流 | 采样电阻 / 霍尔 | 小电阻低损耗 |
| 输入电压 | 电阻分压 | 耐压足够 |
| 温度 | NTC / 热电偶 | 响应速度 |

#### 保护电路

```
过压 → 比较器触发 → 关断PWM → 锁死
过流 → 比较器触发 → 关断PWM → 锁死
过热 → NTC比较 → 关断或降额
```

### 6.3 控制器选择

| MCU | 优势 | 适合 |
|-----|------|------|
| STM32F103 | 资料多、便宜、够用 | 大部分电源题 |
| STM32G4 | 内置运放、比较器、HRTIM | 高精度数字电源 |
| TMS320F28035 | DSP、PWM灵活 | 复杂控制算法 |
| 纯模拟 | TL494/SG3525 | 简单DC-DC |

## 八、比赛策略与调试技巧

### 8.1 时间分配

| 时间段 | 任务 |
|--------|------|
| 第一天上午 | 分析题目、确定方案、分工 |
| 第一天下午~第二天 | 硬件焊接、开环调试 |
| 第三天 | 闭环控制、性能优化 |
| 第四天上午 | 极限测试、记录数据 |
| 第四天下午 | 文档整理、封箱提交 |

### 8.2 排故速查表

| 现象 | 可能原因 | 检查 |
|------|----------|------|
| 没输出 | PWM没工作 | 示波器看驱动引脚 |
| 输出不对 | 反馈分压错 | 万用表测反馈脚电压 |
| 输出振荡 | PI参数不当 | 减小Kp |
| 加载掉电压 | I太小 | 增大Ki |
| 炸管 | 死区不够/占空比太大 | 检查死区、限幅 |
| 纹波大 | LC滤波不足/采样噪声 | 增大电感/电容/滤波 |
| 过热 | 频率不对/没散热 | 检查开关频率 |

### 8.3 调参数口决

1. **先开环再闭环** — 手动给占空比确认功率级正常
2. **先P再I** — Ki=0, 加Kp直到微振, 取60%
3. **再加I** — 缓慢加Ki到消除静差
4. **动静态兼顾** — 突加负载恢复时间 < 5ms
5. **限幅保护** — 占空比限幅 0.95, 积分限幅

### 8.4 备赛建议

1. **赛前必焊一套标准 Buck** — 熟悉完整流程
2. **准备模块化 PCB** — 功率板、控制板分离
3. **备齐关键物料** — MOSFET、电感、电容、驱动芯片
4. **写好串口调参工具** — 实时改参数、看波形
5. **准备示波器** — 没有示波器等于盲调
