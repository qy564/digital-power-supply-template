## 三、参数计算

### 3.1 占空比

$$D = \frac{V_{out}}{V_{in}} = \frac{5V}{12V} = 0.417$$

Vin=24V 时：D = 5/24 = 0.208

占空比范围：**0.21 ~ 0.42**

### 3.2 电感计算

电感纹波电流取负载电流的 30%：

$$\Delta I_L = 0.3 \times I_{out} = 0.3 \times 2A = 0.6A$$

$$\require{enclose} L = \frac{V_{out} \times (1-D_{min})}{f_s \times \Delta I_L} = \frac{5 \times (1-0.208)}{100k \times 0.6} = 66\mu H$$

**选型：** 100μH 功率电感（留余量）

电感峰值电流：$$I_{L\_peak} = I_{out} + \frac{\Delta I_L}{2} = 2 + 0.3 = 2.3A$$

**电感规格：** 100μH / 3A 以上 / 磁屏蔽

### 3.3 输出电容

$$C_{out} = \frac{\Delta I_L}{8 \times f_s \times \Delta V_{out}} = \frac{0.6}{8 \times 100k \times 0.05} = 15\mu F$$

实际取 220μF 电解电容（ESR 主导纹波，要大容量降低 ESR 影响）

### 3.4 输入电容

$$C_{in} = \frac{I_{out} \times D \times (1-D)}{f_s \times \Delta V_{in}}$$

算得约 47μF，实际取 **220μF/50V 电解 + 0.1μF MLCC**

### 3.5 续流二极管

- 反向耐压：> Vin_max = 24V，取 **40V**
- 电流：> Iout = 2A，取 **3A**
- 选 **SS34 肖特基二极管**（40V/3A，VF=0.5V）

### 3.6 MOSFET

- 耐压：> 2×Vin_max = 48V
- 电流：> 3×Iout = 6A
- Rdson：越小越好（< 0.1Ω）
- **选 IRF540N**（100V/33A/0.077Ω），便宜好买

## 五、PCB 布局指南

### 5.1 关键走线原则

```
                    ┌─────────────────────────────┐
                    │         功率回路             │
INPUT ──── Cin ──── Q1 ──── L1 ──── Cout ──── OUTPUT
                     │       │         │
                    GND ◄── D1 ────── GND
                    (功率地)          (功率地)
```

**布局三原则：**

1. **功率回路要短** — Cin → Q1(D) → Q1(S) → L1 → Cout → GND → Cin 围成的回路面积最小化
2. **功率地和控制地单点连接** — 在输入电容 GND 引脚处单点汇合
3. **驱动信号远离功率回路** — 驱动线走背面，不与功率线平行

### 5.2 推荐布局

```
┌───────────────────────────────────────┐
│  ┌─────────┐  ┌───────┐               │
│  │ STM32   │  │       │  Q1  L1       │
│  │ 最小系统 │  │ IR2104├─ D ─┬─┐       │
│  │         │  │ 驱动   │  S  │  │       │
│  │    PA0←──┤  │       │   ──┘  │       │
│  │    PA1←──┤  └───────┘   D1   │       │
│  │         │       │          │        │
│  │  TIM1 ──┤───────┘        Cin Cout   │
│  └─────────┘                         │
│      │                               │
│  ┌───┴────┐  ┌───────┐              │
│  │ LM358  │  │ 保护   │              │
│  │ 比较器  │  │ 电路   │              │
│  └────────┘  └───────┘              │
└───────────────────────────────────────┘
```

## 六、STM32 代码结构

### 6.1 定时器配置

```c
// TIM1 配置 100kHz PWM
void PWM_Init(void) {
    // 预分频: 72MHz / 72 = 1MHz
    // 自动重装: 1000 → 1MHz / 1000 = 1kHz (不对)
    // 自动重装: 720 → 72MHz / 72 = 1MHz / 720 = 1.39kHz?
    // 正确: 72MHz / 720 = 100kHz 周期
    // 预分频 PSC = 0, ARR = 720 - 1
    TIM1->PSC = 0;
    TIM1->ARR = 720 - 1;           // 72MHz / 720 = 100kHz
    TIM1->CCR1 = 300;              // 初始占空比 ~42%
    TIM1->CCMR1 |= TIM_CCMR1_OC1M_1 | TIM_CCMR1_OC1M_2;  // PWM1模式
    TIM1->CCER |= TIM_CCER_CC1E;   // 使能CH1输出
    TIM1->BDTR |= 72;               // 死区 1μs (72个时钟)
    TIM1->CR1 |= TIM_CR1_CEN;      // 启动定时器
}
```

### 6.2 ADC 采样

```c
// 配置 PA0 为 ADC1_IN0
void ADC_Init(void) {
    RCC->APB2ENR |= RCC_APB2ENR_ADC1EN;
    ADC1->SQR3 = 0;                // 第1个转换: 通道0
    ADC1->SMPR2 = 7 << 0;          // 采样周期 239.5周期
    ADC1->CR2 |= ADC_CR2_ADON;     // 开启ADC
    // 校准
    ADC1->CR2 |= ADC_CR2_CAL;
    while(ADC1->CR2 & ADC_CR2_CAL);
}
```

### 6.3 PI 控制器

```c
typedef struct {
    float Kp;
    float Ki;
    float integral;
    float out_min;
    float out_max;
    float integral_limit;
} PID_t;

float PID_Update(PID_t *pid, float setpoint, float feedback) {
    float error = setpoint - feedback;

    // 积分分离：误差大时冻结积分
    if (fabsf(error) < pid->integral_limit) {
        pid->integral += pid->Ki * error;
        // 积分限幅
        if (pid->integral > pid->out_max) pid->integral = pid->out_max;
        if (pid->integral < pid->out_min) pid->integral = pid->out_min;
    }

    float output = pid->Kp * error + pid->integral;

    if (output > pid->out_max) output = pid->out_max;
    if (output < pid->out_min) output = pid->out_min;

    return output;
}
```

### 6.4 主循环

```c
// 控制中断 100kHz (每 10μs)
void TIM1_UP_IRQHandler(void) {
    if (TIM1->SR & TIM_SR_UIF) {
        TIM1->SR &= ~TIM_SR_UIF;

        static uint8_t div = 0;
        div++;
        if (div >= 10) {  // 每 100μs 执行一次控制 (10kHz)
            div = 0;

            // 1. 读取 ADC
            ADC1->CR2 |= ADC_CR2_SWSTART;
            while(!(ADC1->SR & ADC_SR_EOC));
            uint16_t adc_val = ADC1->DR;
            float vout = adc_val * 3.3f / 4096.0f * (12.2f / 2.2f);

            // 2. 保护检查
            if (vout > 6.0f) {  // 过压保护
                TIM1->CR1 &= ~TIM_CR1_CEN;  // 停止PWM
                while(1);  // 锁死
            }

            // 3. PID 计算
            float duty = PID_Update(&pid, 5.0f, vout);

            // 4. 更新 PWM 占空比
            TIM1->CCR1 = (uint16_t)(duty * (720 - 1));
        }
    }
}
```

## 七、调试步骤

### 7.1 硬件检查

```
□ 用万用表测各节点有无短路（电源对地）
□ 上电前确认 Cin 极性正确
□ 先不焊 MOSFET，测驱动波形
□ 确认 IR2104 VB-VS 间电压约 10V（自举正常）
□ 焊上 MOSFET 和电感，测开关节点 SW 波形
```

### 7.2 开环调试

```
1. 烧录程序，给固定占空比 30%
2. 示波器测 Q1 Gate 波形：方波？频率对？幅值够？
3. 示波器测 SW 节点：开关波形有无振铃？
4. 万用表测 Vout：是否约等于 Vin × D？
5. 加 10Ω 负载：电压是否保持？
```

### 7.3 闭环调试

```
1. Ki = 0，Kp 从 0.1 开始逐步增加
2. 观察输出：轻微振荡 → 记下临界 Kp
3. Kp 取临界值的 60%
4. 逐步增加 Ki，直到稳态误差 < 1%
5. 突加/突卸负载：观察恢复时间
```

## 八、常见问题排查

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| 没输出 | PWM 未工作 | 示波器查 TIM1_CH1 引脚 |
| 输出低 | MOSFET 没完全导通 | 查驱动电压够不够 |
| MOSFET 发热 | 驱动不足或频率太高 | 查驱动波形、降低频率 |
| 输出振荡 | PID 参数不当 | 减小 Kp |
| 加载电压掉 | 电感饱和 | 换大电流电感 |
| 输出纹波大 | 电容 ESR 高 | 并联 MLCC |
| 自举失败 | 占空比太大或频率太低 | D < 0.95, fsw > 10kHz |
| 开机冲击大 | 无软启动 | 占空比从 0 逐渐增加到目标值 |

## 九、扩展练习

完成基础 Buck 后，可尝试以下进阶练习：

1. **同步整流** — 用第二个 MOSFET 代替 SS34，效率提升 2~5%
2. **恒流模式** — 添加电流环，实现恒压恒流双环控制
3. **多路输出** — 增加一路 Boost，做一个 Buck+Boost 组合
4. **数字调压** — 通过按键或串口修改目标电压
5. **远程监控** — 串口上报电压电流数据到电脑绘图
